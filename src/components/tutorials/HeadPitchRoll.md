# Head Pitch Roll

[查看链接](https://blog.csdn.net/qq_37796475/article/details/79188477)

## Head
::: tip 
  默认0度为朝向东 数值增大时 顺时针转向 数值减少时逆时针转向

  Head是围绕Y轴旋转，也叫偏航角
:::

## Pitch

::: tip
  默认0度时朝向正前方 数值增大时 仰视  数值减少时 俯视
  
  pitch是X轴旋转，也叫做俯仰角
:::

## Roll

::: tip
  默认0度时朝向正前方 数值增大时 绕x轴 向左旋转  数值减少时 绕x轴 向右旋转

  roll是围绕Z轴旋转，也叫翻滚角
:::
